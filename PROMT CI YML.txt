# 3. Fix the On-Chain CI Workflow

The current workflow contains an invalid GitHub Actions expression similar to:

```yaml
if: ${{ env.ETH_RPC_URL != '' }}
```

or

```yaml
if: env.ETH_RPC_URL != ''
```

GitHub Actions cannot evaluate this expression in that location.

Find every occurrence of this issue and replace it with the correct implementation.

If the workflow expects a GitHub Secret, use the proper `secrets` context.

If it expects a repository variable, use the proper `vars` context.

If necessary, redesign the validation logic so the workflow is fully compatible with GitHub Actions.

The workflow must successfully parse before execution.
